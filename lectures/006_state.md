# state
โดยปกติแล้ว property ที่อยู่ใน component จะถูก track การเปลี่ยนแปรงโดย angular เมื่อพบว่า property มีการเปลี่ยนแปรง angular จะสั่งให้่ render dom นั้นใหม่โดยอัตโนมัติเพื่อทำการอัพเดทข้อมูล

## zone.js
เป็นส่วนหนึ่งของ angular framework โดยมันทำหน้าที่ดักจับเหตุการณ์ของผู้ใช้ที่เป็นไปได้ทั้งหมดโดยอัตโนมัติ
