# Component

    @Component({
        selector: 'app-root',
        standalone: true,
        imports: [],
        templateUrl: './app.html',
        styleUrl: './app.css'
    })

แต่ละ field คือ
- selector: ใช้สำหรับระบุชื่อที่ใช้ในการเรียกใช้งาน component นี้
- standalone:
- imports:
- templateUrl: ระบุส่วนแสดงผลของ component
- styleUrl: ระบุ style sheet ของ component
