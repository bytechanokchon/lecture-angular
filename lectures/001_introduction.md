# Introduction to Angular Framework
เป็น frontend javascript framework ที่รวบรวมเครื่องมือสำหรับการพัฒนาเว็บแอพพลิเคชันไว้ให้เรา
ทำให้สามารถพัฒนาได้ง่ายและเร็วขึ้น

## ทำไมต้องใช้ Framework ?
หากเป็นเว็บแอพพลิเคชันที่มีความซับซ้อนมาก การใช้งาน framework จะทำให้การพัฒนาระบบที่มีความซับซ้อนสูงสามารถพัฒนาได้ง่ายขึ้น
